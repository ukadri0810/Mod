# MOD — Moods of Desserts Premium Demo V4

A responsive, custom-coded demo website built for GitHub Pages.

## Files

- `index.html` — website structure and editable content
- `styles.css` — complete visual design and responsive layouts
- `script.js` — loading animation, cinematic hero, menu, filters and reveals
- `assets/` — local optimized food and logo images

## Publish on GitHub Pages

1. Create or open your GitHub repository.
2. Upload everything inside this folder to the repository root.
3. Open **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select `main` and `/ (root)`, then save.
